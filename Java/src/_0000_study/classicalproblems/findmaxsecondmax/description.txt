
寻找数组中的最大值和次大值：


思想如下：用例子演示:
    nums : [3, 2, 1, 5]
    寻找最大值：从低向上，两两对比
                5
              /   \
            3*     5
          /  \    /  \
         3    2  1*   5
    寻找次大值：从顶向下，和最大值比较过的数据可能是次大值，找出和最大值比较过的数据中的最大值就是次大值


    需要重新开辟空间：2n-1
    [5, 3, 5, 3, 2, 1, 5]
     -------  ----------
                原始数据
