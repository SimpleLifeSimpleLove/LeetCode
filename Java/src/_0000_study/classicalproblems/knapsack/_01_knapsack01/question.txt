0-1背包问题：
    有一个背包，它能承受的最大重量为C（Capacity）。现在由n种不同的物品，编号为0...n-1，
    其中每一件物品的重量为weights[i]，价值为values[i]，可以放入背包，也可以不放入。
    问可以向这个背包中盛放哪些物品，使得在不超过背包容量的基础上，物品的总价值最大。

暴力解法：每一件物品都可以放进背包，也可以不放入背包：O((2^n) * n)

动态规划解法：
    状态：dp(n, C) 考虑将前n个物品放进容量为C的背包，使得价值最大
    状态转移方程：dp(i, c) = max(dp(i-1, c), dp(i-1, c - w(i)) + v(i))
    具体实现：自上而下（记忆化搜索），自下而上（动态规划）



拓展：
如果要求恰好填满容量为 C 的背包，则改变初始化就行(Knapsack01_dp4)
        Arrays.fill(dp, Integer.MIN_VALUE);
        dp[0] = 0;
如果最终dp[C] <= 0 说明，没有解，不能恰好填满!!!
