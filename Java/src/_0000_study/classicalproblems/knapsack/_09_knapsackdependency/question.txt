有依赖的背包问题：
    有一个背包，它能承受的最大重量为C（Capacity）。现在由n种不同的物品，编号为0...n-1，
    其中每一件物品的重量为weights[i]，价值为values[i]，可以放入背包，也可以不放入。
    如果选择一个物品，则必须选择其依赖的物品。问可以向这个背包中盛放哪些物品，使得在不超过
    背包容量的基础上，物品的总价值最大。输出最大价值。数据保证是一棵树。

    父节点编号范围：0<= pi <= n-1       根节点：pi = -1

输入样例：
    weights     values      parent
       2          3           -1
       2          2           0
       3          5           0
       4          7           1
       3          6           1

状态：dp[i][j] : 背包容量为j的情况下，选择i号物品(如果容量允许)，最大的收益
