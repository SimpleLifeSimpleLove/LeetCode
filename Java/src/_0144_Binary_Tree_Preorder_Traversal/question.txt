Leetcode 0144 二叉树的前序遍历

// 给你二叉树的根节点 root ，返回它节点值的 前序 遍历。
//
// 示例 1：
// 输入：root = [1,null,2,3]
// 输出：[1,2,3]
//
// 示例 2：
// 输入：root = []
// 输出：[]
//
// 示例 3：
// 输入：root = [1]
// 输出：[1]
//
// 示例 4：
// 输入：root = [1,2]
// 输出：[1,2]
//
// 示例 5：
// 输入：root = [1,null,2]
// 输出：[1,2]
//
// 提示：
// 树中节点数目在范围 [0, 100] 内
// -100 <= Node.val <= 100
//
// 进阶：递归算法很简单，你可以通过迭代算法完成吗？
// Related Topics 栈 树
// 👍 496 👎 0


解法说明：
    Solution：经典递归算法，调换顺序可直接改写成为中后序遍历
    Solution2：模拟系统栈，非递归递归算法，调换顺序可直接改写成为中后序遍历
    Solution3：经典非递归算法
    Solution4：另一种经典非递归算法
    Solution5：另一种经典非递归算法
    Solution6：莫里斯前序遍历，优点：空间复杂度为O(1)
